prx_nu_from_t_a¶

pycraf.conversions.prx_nu_from_t_a(T_A)[source] [edit on github]¶

Received spectral power from antenna temperature.

The received spectral power is the power per unit frequency, i.e., prx_nu = prx / bandwidth.

Parameters:
T_a : Quantity

Antenna temperature [K]
Returns:
prx_nu : Quantity

Received spectral power [W/Hz]

Notes

Because all parameters/returned values are Astropy Quantities (see Quantity), unit conversion is automatically performed.
