gain_from_eff_area¶

pycraf.conversions.gain_from_eff_area(eff_area, freq)[source] [edit on github]¶

Antenna gain from effective antenna area, given frequency.

Parameters:
eff_area : Quantity

Effective antenna area, A_eff [m**2]

freq : Quantity

Frequency [Hz]
Returns:
gain : Quantity

Antenna gain [dBi]

Notes

Because all parameters/returned values are Astropy Quantities (see Quantity), unit conversion is automatically performed.
